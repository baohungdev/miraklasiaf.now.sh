```bash

$ vc secret add firebase-client-email
```

```bash:.env
FIREBASE_SECRET_KEY=xxxxx
FIREBASE_CLIENT_EMAIL=xxxxx
```

Finally, we need to add our new secret to `vercel.json`.

```json:vercel.json
{
  "env": {
    "FIREBASE_SECRET_KEY": "@firebase-secret-key",
    "FIREBASE_CLIENT_EMAIL": "@firebase-client-email"
  }
}
```

We can now reference the secrets via `process.env.FIREBASE_SECRET_KEY` and `process.env.FIREBASE_CLIENT_EMAIL`.
